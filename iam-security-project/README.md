#AWS IAM Security Mini Project

## Overview
This project demonstrates AWS Identity and Access Management (IAM) best practices
by creating a non-root IAM user with least-privilege permissions.

## AWS Services Used
- AWS IAM

## Key Concepts Demonstrated
- Root vs IAM users
- Least privilege access
- IAM policies
- Multi-factor authentication (MFA)

## What I Learned
- How to securely manage access in AWS
- Why MFA is important for cloud security
