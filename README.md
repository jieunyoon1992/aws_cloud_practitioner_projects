# AWS S3 Static Website Hosting Project

## Overview
This project demonstrates hosting a static website using Amazon S3 as part of my AWS Cloud Practitioner preparation.

## AWS Services Used
-Amazon S3
-AWS IAM
-AWS Management Console

## Architecture
-Static HTML file hosted in an S3 bucket
-Public read access enabled via bucket policy
-Website accessed through S3 static website endpoint

## Key Concepts Demonstrated
-Cloud storage using Amazon S3
-Public vs private access control
-AWS global regions
-Shared Responsibility Model
-Free Tier usage awareness

## Screenshots
Screenshots of bucket configuration, permissions, and website access are included.

## What I learned
-How to configure S3 static website hosting
-How IAM and bucket policies affect security
-Best practices for managing public access in AWS
